# Deployment Guide

1. Create a GitHub repository named exactly shovon-mandal.github.io.
2. Upload all files from this package to the repository root.
3. Go to Settings > Pages.
4. Select Deploy from a branch.
5. Select main branch and /root folder.
6. Save and wait a few minutes.
7. Open https://shovon-mandal.github.io.

## Updating later
Use admin.html to download a new site-data.json, then replace data/site-data.json in GitHub.

## CV link
In your CV header, write Website and hyperlink it to https://shovon-mandal.github.io.
