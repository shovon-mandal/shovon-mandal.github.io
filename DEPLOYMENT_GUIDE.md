# Deployment Guide

Upload all files from this folder to the root of your GitHub Pages repository.

Required structure:
- index.html
- assets/css/style.css
- assets/js/main.js
- assets/img/profile.jpg
- assets/docs/Shovon_Mandal_CV.pdf
- data/site-data.json
- README.md
- DEPLOYMENT_GUIDE.md
- .nojekyll

Then run:

git add -A
git commit -m "Publish final academic portfolio"
git push
