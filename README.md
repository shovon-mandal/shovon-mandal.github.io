## Fresh GitHub Pages Upload Instructions

If you want to delete everything from the existing website repository and upload this fresh version:

1. Download and unzip this package.
2. Open the unzipped folder.
3. Delete all old files from the GitHub Pages repository.
4. Copy all files from this package into the repository root.
5. Commit and push using the commands below.

```bash
git add -A
git commit -m "Publish Shovon Mandal academic portfolio"
git branch -M main
git remote set-url origin https://github.com/shovon-mandal/shovon-mandal.github.io.git
git push -u origin main
```

If the repository has no remote yet, use this instead of `git remote set-url`:

```bash
git remote add origin https://github.com/shovon-mandal/shovon-mandal.github.io.git
git push -u origin main
```
